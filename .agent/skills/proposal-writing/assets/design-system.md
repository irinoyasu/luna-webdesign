# 日本語コンサルデザインシステム・リファレンス (Japanese Consulting Design System)

プロポーザルおよびプレゼンテーション（GASによるスライド生成等）を作成する際は、以下のデザイン原則とアンチパターンを必ず遵守してください。

## 1. デザイン・アンチパターン（避けるべきこと）
- ❌ **トピックラベルのみの汎用タイトル**（アクションタイトルでないもの）の使用は厳禁です。
- 各スライド、各セクションには必ず**アクションタイトル**（Action Title）を付けること。単なるトピックの名称ではなく、「そのスライド/セクションで伝えたい洞察や結論を述べる完全な文」としてください。
  - ✅ **アクションタイトルの例**: 「デジタル売上は前年比34%増、主にエンタープライズ契約が牽引」
  - ❌ **トピックラベルの例（NG）**: 「売上概要」

## 2. 日本語特有のフォーマット・注意点
高品質なビジネスドキュメント・スライドを出力するため、以下のルールを厳守してください。

- **フォントの指定**:
  - 日本語見出し: `游明朝 (Yu Mincho)` または `MS 明朝` を使用する。
  - 本文: `游ゴシック (Yu Gothic)` または `Meiryo UI` を使用する。
  - 英数字・データ数値: `Calibri` を使用する。
- **文字サイズ**:
  - 日本語は英語フォントよりも小さく見えがちなため、英語ベースのテンプレートから出力する場合は **1-2pt大きくして** 可読性を確保すること。
- **テキストボックスの幅**:
  - 日本語テキストは英語の同一情報量と同等かそれ以上の幅が必要になることが多い。意図しない改行を防ぐため、余裕を持った幅（width）設定を行うこと。
- **句読点**:
  - 全角の「。」「、」を使用すること（半角カンマやピリオドを日本語文中に混ぜない）。
- **敬語レベル・トーン＆マナー**:
  - ビジネス文書として適切な敬語レベルを維持する。「です・ます」調を基本とする。
  - 顧客を主語とし、顧客に対する敬意と自信を感じさせるプロフェッショナルなトーンとすること。
- **数字表記**:
  - 大きな数字は漢数字（億、万）と算用数字を適切に組み合わせ、パッと見で金額感を理解できるようにする。
  - ✅ 良い例：`1.2兆円`、`340億円`
  - ❌ 悪い例：`1,200,000,000,000円`、`34,000,000,000円`
